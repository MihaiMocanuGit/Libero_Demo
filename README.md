# Libero_Demo

Libero_Demo is both an example on how to use and an experimentation sandbox for the Libero library.

Building and running the demo is done using the bash script: `./run.sh`. Run `./run.sh -h` for more
details.

Note that it's not necessary to use Libero as a git submodule. It has been done this way in the case
of Libero_Demo so that Libero is easily modified and updated while experimenting here.
